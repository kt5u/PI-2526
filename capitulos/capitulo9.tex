\chapter{Interpretação dos Resultados dos Questionários}

Após a recolha e submissão dos questionários, realizou-se uma análise detalhada das respostas obtidas. Para esse efeito, recorreu-se ao processo ETL (Extract, Transform, Load), no qual os dados provenientes do Google Forms foram inicialmente exportados no formato CSV e, posteriormente, sujeitos a procedimentos de tratamento e transformação, de forma a garantir a sua organização, consistência e adequação para análise.

\section{Processo ETL}
O processo ETL foi dividido em três etapas principais:

\begin{itemize}
  \item \textbf{Extract:} Os dados foram extraídos do Google Forms e exportados para um ficheiro CSV.
  \item \textbf{Transform:} Os dados foram limpos e transformados para garantir a sua consistência e adequação para análise. Isso incluiu a remoção de respostas duplicadas, a correção de erros de entrada e a normalização de formatos.
  \item \textbf{Load:} Os dados transformados foram carregados em um sistema de análise, onde foram realizados testes estatísticos e análises descritivas para interpretar os resultados.
\end{itemize}
