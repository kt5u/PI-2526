\chapter{Conclusão}
\label{cap6}

Ao longo dos três anos letivos, o projeto IPB Recicla evoluiu significativamente, passando de uma aplicação funcional básica para uma plataforma robusta, com estrutura de dados otimizada, cálculos ambientais validados e interface moderna. Cada ciclo contribuiu com avanços técnicos e funcionais, consolidando a aplicação como uma ferramenta educativa e ambiental relevante para a comunidade do IPBeja.
